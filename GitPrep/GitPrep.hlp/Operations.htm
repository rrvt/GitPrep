GitPrep Operations

Specify Git Directory

Specify the Git Directory that holds one or more uploadable Project Directories. For example:

D:\Git\GitPrep

Find Redundant Files

The application holds lists of redundant directories and file extensions of redundant files. This command will display all files found that the application considers to be redundant.

Commit to Delete Files Shown

Commit to deleting all the files listed.

Find Large Files

Sometimes large files have dificulty being uploaded to Git. This command will list large files in the upload. Please examine the files carefully before either deleting them or executing the Commit command.

Save File As

Displays a Save File Dialog and saves information displayed to the file named.

Print

Will do all the normal things to print displayed information.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.
